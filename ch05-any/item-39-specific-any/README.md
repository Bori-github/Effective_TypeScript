# TL;DR
- `any` 타입은 자바스크립트에서 표현할 수 있는 모든 값을 아우르는 큰 범위의 타입이기 때문에 사용하기 전에 정말로 모든 값이 허용되어야 하는지 검토해야합니다.
- 일반적인 상황에서는 `any` 보다 더 구체적으로 표현할 수 있는 타입이 존재할 가능성이 높습니다.
- 따라서, 타입의 안정성을 높이기 위해 보다 구체적인 형태의 타입을 사용해야 합니다.
