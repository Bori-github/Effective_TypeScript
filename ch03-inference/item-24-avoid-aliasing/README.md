# TL;DR
- 어떤 값을 새로운 변수(별칭)에 할당하여 사용하기도 합니다. 이러한 별칭을 남용하게 된다면 제어 흐름 분석이 어려워집니다.
- 속성 체크를 통해 타입 정제를 하는 경우 별칭을 일관되게 사용하지 않는다면 타입 오류가 발생할 수 있습니다.
- 일관된 별칭을 사용하기 위해 비구조화 문법을 사용하는 것이 좋습니다.
