# TL;DR
- 자바스크립트에서 하나의 변수를 다른 목적을 가지는 다른 타입으로 재사용 할 수 있습니다.
- 타입스크립트에서는 다른 타입으로 인해 오류를 나타내지만 유니온 타입과 같은 기능을 활용하여 타입을 변경한 후 재사용 할 수 있습니다.
- 하지만 서로 관련없는 값은 분리하는 것이 좋습니다. 따라서, 다른 타입에는 다른 변수를 선언하여 사용하는 것이 바람직합니다.
